# 하루홈 가계부

브라우저에서 바로 사용할 수 있는 심플한 가계부 웹 앱입니다. 별도의 서버나 빌드 도구 없이 `index.html`만 열면 동작합니다.

## 주요 기능

앱은 세 개의 탭으로 구성됩니다: **거래 · 자산 · 목표**

### 거래 탭
- 수입/지출 거래 기록 (날짜, 카테고리, 금액, 메모)
  - 수입 카테고리: 급여(노현준), 급여(김혜민), 용돈, 보너스, 이자/투자, 환급, 기타
  - 지출 카테고리: 식비, 카페/간식, 교통, 주거/통신, 생활용품, 의료/건강, 문화/여가, 쇼핑, 경조사, 육아, 기타
- 이번 달 **수입 · 지출 · 잔액 · 저축률** 요약 (저축률 = (수입-지출)/수입)
- 카테고리별 지출 비율 분석
- 월별 수입·지출 오버뷰 (최근 12개월, 행 클릭 시 해당 월로 필터링)
- 월 필터 및 전체 보기

### 자산 탭
- 자산/부채 항목 등록 및 총자산·총부채·순자산 요약
- 매달 스냅샷 저장으로 최근 12개월 순자산 추이 막대 차트

### 목표 탭
- **연간 저축 목표** 설정 (연도별 관리, 2026년 기본)
- 누적 저축 / 목표 / 남은 금액 / 달성률 진행 바
- **월별 저축률** 목록

### 기타
- **Google 계정 로그인 → Firebase Firestore 실시간 동기화** (PC ↔ 폰 자동 연동)
- JSON 내보내기 (거래 + 자산 + 추이 + 목표 통합 백업)
- 전체 데이터 초기화
- 로그인하지 않으면 브라우저 `localStorage` 에만 저장 (기기 간 동기화 없음)

## 사용 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 끝!

## 파일 구조

```
.
├── index.html   # UI 마크업
├── styles.css   # 스타일
├── app.js       # 상태 관리 및 렌더링 로직
└── README.md
```

## 기기 간 동기화

앱 상단의 **"Google로 로그인"** 버튼을 누르면, 입력한 데이터가 Firebase Firestore에 자동 저장됩니다. 같은 구글 계정으로 다른 기기에서 로그인하면 데이터가 실시간으로 동기화됩니다.

- 로그인 후 기존 로컬 데이터는 첫 동기화 시 자동으로 클라우드에 업로드됩니다.
- 클라우드에 이미 데이터가 있으면 그 데이터가 현재 기기로 내려옵니다.
- 네트워크가 끊긴 상태에서도 로컬 저장은 계속되며, 연결이 회복되면 자동 동기화됩니다.

## 주의

- 로그아웃 상태에서는 `localStorage` 에만 저장되므로, 기기를 바꾸거나 브라우저 데이터를 초기화하면 사라집니다.
- 필요한 경우 `JSON으로 내보내기` 버튼으로 수동 백업할 수 있습니다.
